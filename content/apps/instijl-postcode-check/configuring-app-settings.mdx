## Adjusting app preferences

The app offers various configuration options. Here is an overview of the settings:

### Address validation - General Settings

These are the basic settings for address validation.

#### Validation and Correction Method

This setting determines how addresses are validated and corrected.

**Correction by Street name and House number:** Select this option to validate and correct addresses based on the street name and house number. This is recommended when postal codes are frequently incorrect.

**Correction by Postal code and House number:** Choose this option to validate and correct addresses using the postal code and house number. This is the best choice if street or city names are frequently incorrect.

**Correction by Postal code and House number, then by Street name and house number:** Choose this option to validate addresses based on both methods.

**Correction by Street name and house number, then by Postal code and House number:** Choose this option to validate addresses based on both methods.

### Address validation - Shipping Address

Here you will find the settings for validating and correcting shipping addresses.

#### Automatic validation of shipping addresses for new orders

Enable this option to automatically validate shipping addresses. Shipping addresses within the Netherlands are validated within seconds right after a new order is placed.

#### Automatic correction of invalid shipping addresses for new orders

Enable this option to automatically correct invalid shipping addresses. Shipping addresses within the Netherlands are corrected within seconds right after a new order is placed.

### Address validation - Billing Address

Here you will find the settings for validating billing addresses.

#### Automatic validation of billing addresses for new orders

Enable this option to automatically validate billing addresses. Billing addresses within the Netherlands are validated within seconds right after a new order is placed.
