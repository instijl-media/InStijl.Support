### How does InStijl Postcode Check work?

InStijl Postcode Check has an (API) connection with Postcode.eu. A subscription to Postcode.eu is therefore required. You can easily request this through this app, or create an account yourself at [www.postcode.eu](http://www.postcode.eu). If you have InStijl Media request a subscription, all you have to do is fill in the company details and we will set up the rest for you! If you request a subscription from Postcode.eu yourself, all you have to do is fill in the API Keys. 

### Does this app work for addresses in countries other than the Netherlands?

Currently, the app only works for Dutch postal codes.

### Why is a subscription to [Postcode.eu](http://Postcode.eu) required?

[Postcode.eu](http://Postcode.eu) ensures that the address database remains constantly up-to-date. We need this data to keep the app working properly.

### Can I test the app first (for free)?

Yes, you can! The app has a 14-day trial in which you can test the app for free. In addition, [Postcode.eu](http://Postcode.eu) offers 200 free address checks when creating an account. That's enough to test the app first without being stuck with a subscription.
