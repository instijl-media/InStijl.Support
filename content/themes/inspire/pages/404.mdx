The 404 page is the error page that visitors see when they click on a link that no longer exists or type in a wrong URL. A good 404 page is helpful and quickly guides the visitor back to the active parts of your webshop.

### General Layout & Design

Through the main settings, you manage the general display of this section. By default, a large question mark icon is shown above the content to make it immediately clear that something is missing.

- **Color scheme:** Choose the desired background and text color for the error page.
- **Section type:** Determine whether the background spans the full width of the screen (**Full width / Edge-to-edge**) or is centered within a frame (**Box / Container**).
- **Custom Class:** An option for advanced users who want to add extra, custom CSS styling to this specific page.

### Spacing (Whitespace)

Ensure the error message is displayed neatly and centered by adjusting the whitespace:

- **Padding (Inner space):** Adjust the distance at the top (**PT**) and bottom (**PB**) inside the section. Choose between the default spacing of the theme or remove the space completely (None).
- **Margin (Outer space):** Determine the distance at the bottom (**MB**) outside the entire section.

### Content and Navigation (Blocks)

You build the text and navigation below the question mark icon using standard theme blocks. It is highly recommended to give the visitor a helping hand here so they don't get stuck:

- **Title and Text:** Add standard text blocks to briefly explain what is going on (e.g., "Oops! Page not found" or "The page you are looking for no longer exists.").
- **Buttons:** Add a clear button (e.g., "Back to Home" or "View our products") that brings the visitor directly back to a working part of your webshop.
