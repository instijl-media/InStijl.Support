# InStijl AI

Welkom bij de documentatie van **InStijl AI**, een AI-gestuurde productverrijkings-app voor Lightspeed eCom.

## Wat doet het

- Genereert rijke productomschrijvingen
- Extraheert en koppelt specs en attributen
- Stelt nieuwe attribuut-concepten voor wanneer ze in jouw shop ontbreken

Zie [Aan de slag](./getting-started.mdx) voor de installatiestappen.
